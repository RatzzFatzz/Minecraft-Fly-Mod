# Minecraft - Fly-Mod
This Mod for Minecraft allows you to enable a 3D fly and let you customize the fly and run speed.

## Requirements
* Any of the listed Minecraft versions
    * 1.14.4
    * 1.15.1 & 1.15.2
    * 1.16.1 - 1.16.5
    * 1.17 - 1.17.1
* Fabric
* ModMenu
* Java 8 (Java 16 for Minecraft 1.17 and later)

## Supported languages
* Deutsch
* English

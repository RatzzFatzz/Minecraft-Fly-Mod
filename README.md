# Minecraft - Fly-Mod
This Mod for Minecraft allows you to enable a 3D fly and let you customize the fly and run speed.

## Requirements
* Any of the listed Minecraft versions
    * 1.14.4
    * 1.15.1
    * 1.15.2
    * 1.16.1
    * 1.16.2
    * 1.16.3
* Fabric
* ModMenu
* Java 8

## Supported languages
* Deutsch
* English
